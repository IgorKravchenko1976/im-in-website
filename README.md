# Im-in - Сайт-візитка проєкту

Сучасний односторінковий сайт-візитка для додатку Im-in - платформи для відмічання подій на карті, обміну фото та відео, створення подій та спілкування.

## 🌟 Особливості

- **Сучасний дизайн** - градієнти, анімації та плавні переходи
- **Адаптивна верстка** - працює на всіх пристроях (десктоп, планшет, мобільний)
- **Багатомовність** - підтримка 7 мов: Українська, Англійська, Французька, Іспанська, Німецька, Італійська, Грецька
- **Інтерактивність** - плавна прокрутка, анімації при скролі, мобільне меню
- **Оптимізація** - швидке завантаження та оптимізований код
- **SEO-friendly** - правильна структура HTML та мета-теги

## 📁 Структура проєкту

```
IM-IN/

├── styles.css          # Стилі та дизайн
├── script.js           # JavaScript функціональність
```

## 🚀 Запуск сайту

### Локальний запуск

1. Відкрийте файл `index.html` у браузері
2. Або використайте локальний сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (якщо встановлений http-server)
npx http-server

# PHP
php -S localhost:8000
```

Потім відкрийте `http://localhost:8000` у браузері.

### Деплой на хостинг

Для домену `im-in.net`:

1. Завантажте всі файли на ваш хостинг
2. Налаштуйте DNS записи для домену
3. Переконайтеся, що `index.html` знаходиться в кореневій директорії

## 🎨 Кастомізація

### Зміна кольорів

У файлі `styles.css` знайдіть секцію `:root` та змініть змінні:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... інші кольори */
}
```

### Зміна контенту

Відредагуйте файл `index.html` для зміни тексту, додавання нових секцій або зміни структури.

### Додавання анімацій

Анімації налаштовані в `styles.css` та `script.js`. Ви можете додати нові або змінити існуючі.
